# ElementaryOS Tweaks

Add PPA Support
```
sudo apt install software-properties-common
```

Install Elementary Tweaks
```
sudo add-apt-repository ppa:philip.scott/elementary-tweaks
sudo apt update
sudo apt install elementary-tweaks
```
